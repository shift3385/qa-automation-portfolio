We are seeking a skilled Automated QA Engineer to join our quality assurance team in ensuring the delivery of high-quality software solutions for both our internal and external customers. In this role, you will be responsible for designing, developing, and maintaining automated tests for web applications using Cypress, as well as automated API tests using Postman and/or SOAPUI. You will play a key role across the software development life-cycle, collaborating with cross-functional teams to ensure products meet functional and quality requirements.

Responsibilities

Develop, enhance, and maintain automated test scripts for web applications using Cypress.
Design and execute automated API tests using Postman and/or SOAPUI for REST and/or SOAP services.
Work closely with developers, business analysts, and other stakeholders to review requirements and ensure comprehensive test coverage.
Participate in all phases of the software development life cycle, from requirements analysis to release.
Execute and monitor automated regressions and report results, defects, and quality trends.
Troubleshoot test failures, analyze logs, and work with the development team to resolve issues.
Maintain clear documentation of test cases, scripts, and test results.
Assist in building and improving continuous integration pipelines for test automation execution.
Participate in code reviews related to automated tests to ensure best practices.
Communicate test progress, risks, and issues to stakeholders in a timely and effective manner.
Continuously improve skills and keep up with emerging automation frameworks and tools.

Qualifications

Proven experience in automated quality assurance for web applications.
Proficient in developing automated tests using Cypress.
Experience designing and executing automated API tests using Postman and/or SOAPUI.
Understanding of REST and SOAP services.
Familiarity with continuous integration tools such as Jenkins, GitHub Actions, or similar.
Solid experience with defect tracking tools and test management systems.
Strong analytical, troubleshooting, and problem-solving skills.
Detail-oriented with a focus on producing high-quality deliverables.
Ability to clearly document test cases and results.
Effective communication skills in English, both oral and written.
Experience working collaboratively in Agile/Scrum teams.
Bachelor’s degree in Computer Science, Engineering, or a related field preferred.
